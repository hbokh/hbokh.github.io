macOS Weather widget reborn

May 17, 2020 #apple

Ever since using Mac OS X for the first time, I’ve been a sucker for the Dashboard: pressing F4 on any Mac has become muscle memory over the last 14 years.

Somewhere in the recent past Mac OS X was renamed to macOS and the F4 key was given a different function: Launchpad.
And instead of Dashboard, we’re supposed to use Notifications…

So Apple’s love for the Dashboard and its widgets had faded. But not mine!
I’ve always tried to reprogram the F4 key to show the Dash anyhow.

In the fall of 2019 the Weather widget stopped working: the main widget I was pressing F4 for, for all those years.

But recently (May 2020) a forum post on MacRumors made my day.
It’s been there for a couple of weeks, but by the end of last April the member by the name of “Wowfunhappy” finally managed to create a working Weather widget.

Read all about it here: I fixed Apple’s broken Weather Dashboard Widget.

After creating the accounts for Weatherbit and MapQuest and editing the file weatherParser.js, the Weather widget was there again, in all its glory! ðŸ¥³
